===============================

Para jogar, rode o arquivo   >>>>> jogador1.py
Após a inicialização, rode o arquivo   >>>>>>> jogador2.py

O arquivo >>>> tabuleiro.py <<<<  é necessário para rodar ambos os arquivos, 
se for testar o arquivo >>>> jogador2.py <<<<  em uma máquina diferente, 
é necessário ter o >>>> tabuleiro.py <<<<  também.

===============================

A execução de partidas são ilimitadas enquanto ambos os jogadores quiserem continuar.

Há a opção de jogos contra o computador.
